Implications for the arc (see [[NOTES.md]]):

- Treat **function extensionality** as the primary motivator for cubical type theory: an axiom in book-HoTT that does not compute, a one-line computing theorem in cubical. This is a concrete mission payoff (it makes "why cubical exists" tangible).
- Frame UIP as "every type is a set" and tie it to the h-levels lesson.

Signal: this learner reaches for the surrounding conceptual landscape, not just the linear path. Lessons can afford to name adjacent principles and place them on the map, not only teach the next step.
